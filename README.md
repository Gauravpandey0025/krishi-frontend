# 🌾 Crop Prediction Frontend

## 🚀 Overview

This is the frontend of the Crop Recommendation System. It allows users to input soil and environmental parameters and get crop recommendations from the backend API.

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

## 🎯 Features

* User-friendly input form
* API integration with Flask backend
* Responsive design
* Displays predicted crop result

## ⚙️ Installation

```bash
git clone https://github.com/your-username/crop-frontend.git
cd crop-frontend
```

## ▶️ Run

Simply open:

```
index.html
```

OR use Live Server in VS Code.

## 🔗 API Integration

Make sure backend is running:

```
http://localhost:5000
```

Update API URL in JS if needed.
